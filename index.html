Setup Instructions

1

Create GitHub Repo

Upload both worker.js and index.html to your GitHub repository.

2

Connect to Cloudflare

In Cloudflare Dashboard, go to Workers & Pages → Create application → Connect to Git.

3

Deploy Worker

Select worker.js as the entry point and deploy. Note your Worker URL.

4

Configure Player

Enter your Worker URL in the "Cloudflare Worker URL" field above.
